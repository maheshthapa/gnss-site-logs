GNSS Site Logs Repository
=========================

DIRECTORY & FILE STRUCTURE:

• site-logs/
  Contains the most recent and actively updated GNSS site log files.

• old-site-logs/
  Archive folder containing legacy/superseded site log files. 
  Note: Always refer to 'site-logs/' for the current operational station records.

• station_list.txt
  A plain text file listing all monitored GNSS stations.
